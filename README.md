# React TalkBot

Convey your message to your users through displaying a bautiful animation

## Installation

```
$ npm i react-talkbot
```

## Usage

```
    import { TalkBox } from 'react-talkbot';

    const App = () => {
        return (
            <div>
                // some code here
                <TalkBox />
            </div>
        )
    }
```
